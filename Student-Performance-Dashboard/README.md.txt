# 🎓 Student Performance Dashboard (Power BI)

## 📌 About the Project

This is my first Power BI dashboard project, created using the Student Performance Dataset from Kaggle. The main goal of this project was to practice data visualization and build an interactive dashboard that helps analyze student performance from different perspectives.

Using Power BI, I explored the data and created meaningful visualizations to understand student marks, attendance, study habits, internet access, and school-wise performance.

---

## 📊 Dashboard Highlights

- Total Students
- Average Final Marks
- Highest Final Marks
- Average Absences
- Average Final Marks by School
- Average Final Marks by Age
- Study Time vs Final Marks
- Gender Distribution
- Students with Internet Access
- Average Absences by School

---

## 🛠 Tools & Technologies

- Microsoft Power BI Desktop
- CSV Dataset (Kaggle)
- GitHub

---

## 📂 Dataset

The dataset used in this project is the **Student Performance Dataset** downloaded from **Kaggle**.

---

## 📈 Key Insights

- Compared student performance between different schools.
- Analyzed how study time relates to final marks.
- Visualized the gender distribution of students.
- Identified the highest final marks achieved.
- Compared average absences across schools.
- Explored internet access among students.
